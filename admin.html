업로드한 이미지·후기·파트너 정보를 JSON 파일로 다운받아 다른 컴퓨터나 브라우저로 옮길 수 있습니다. 저장한 JSON 파일을 다시 불러오면 그대로 복원됩니다.

사용 안내. 업로드한 이미지는 현재 브라우저에만 저장됩니다(localStorage). 실제 서버 연동 전 데모·프로토타입 용도이므로, 작업이 끝나면 반드시 JSON으로 내보내기를 해두세요.

물류·식자재 파트너 로고

협력사 로고와 이름을 등록하면 홈페이지에 "함께하는 물류 파트너" 섹션이 자동으로 표시됩니다. (예: 동원홈푸드, CJ프레시웨이, 현대그린푸드 등) 최대 6개까지 등록 가능. 등록된 파트너가 없으면 해당 섹션은 자동으로 숨겨집니다.

후기 관리

홈페이지 후기 섹션에 흐르는 리뷰를 관리합니다. 6개 이상 등록하면 자연스럽게 무한 순환됩니다.
